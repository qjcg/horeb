# Horeb [![Build Status][build-logo]][horeb-travis]

![Mt. Horeb][mt-horeb]

Speaking in tongues via stdout.

Somewhat inspired by the [TempleOS](http://templeos.org) [oracle].

## Install

```sh
go get -u github.com/qjcg/horeb
```
